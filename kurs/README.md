# Интернет-магазин цифровой техники

Django интернет-магазин цифровых товаров с полным функционалом.

## Возможности

### Для покупателей:
- 📦 Каталог товаров с категориями и фильтрами
- 🔍 Поиск по товарам
- 🛒 Корзина покупок
- 💳 Оформление заказа с выбором доставки и оплаты
- ❤️ Список желаний (Избранное)
- ⭐ Отзывы и рейтинги товаров
- 📋 История заказов

### Для администраторов:
- 📊 Управление товарами, категориями, заказами
- 👥 Управление пользователями
- ⭐ Модерация отзывов
- 📈 Отслеживание статусов заказов

## Установка и запуск

### 1. Установите зависимости (если есть requirements.txt)
```bash
pip install django pillow djangorestframework
```

### 2. Создайте миграции и примените их
```bash
python manage.py makemigrations
python manage.py migrate
```

### 3. Создайте суперпользователя (администратора)
```bash
python manage.py createsuperuser
```
Следуйте инструкциям для создания учетной записи.

### 4. Запустите сервер разработки
```bash
python manage.py runserver
```

### 5. Откройте в браузере
- **Главная страница (каталог):** http://127.0.0.1:8000/
- **Админ-панель:** http://127.0.0.1:8000/admin/

## Модели данных

### Products (Товары)
- **Category** - Категории товаров
- **Product** - Товары с поддержкой цифровых продуктов
- **ProductImage** - Дополнительные изображения товаров

### Users (Пользователи и заказы)
- **Cart** - Корзина покупок
- **CartItem** - Элементы корзины
- **Order** - Заказы
- **OrderItem** - Элементы заказа
- **Wishlist** - Список желаний
- **Review** - Отзывы о товарах

## Настройки

### Основные настройки в `kurs/settings.py`:
- `DEBUG = True` - режим разработки
- `SECRET_KEY` - ключ безопасности (измените для production!)
- `DATABASES` - SQLite база данных
- `STATIC_URL`, `MEDIA_URL` - настройки статических и медиа файлов

### Для production:
1. Установите `DEBUG = False`
2. Настройте `ALLOWED_HOSTS`
3. Используйте PostgreSQL или MySQL вместо SQLite
4. Настройте статические файлы через веб-сервер (Nginx/Apache)
5. Используйте HTTPS

## Добавление товаров

1. Войдите в админ-панель: http://127.0.0.1:8000/admin/
2. Перейдите в раздел "Товары" → "Категории"
3. Создайте категории (например: "Смартфоны", "Ноутбуки", "Планшеты")
4. Перейдите в раздел "Товары" → "Товары"
5. Создайте товары, заполнив все необходимые поля

### Для цифровых товаров:
- Отметьте галочку "Цифровой товар"
- Загрузите файл товара в поле "Файл товара"

## Тестирование функционала

1. **Каталог товаров:**
   - Откройте главную страницу
   - Попробуйте поиск и фильтрацию по категориям

2. **Детальная страница товара:**
   - Кликните на любой товар
   - Посмотрите описание, характеристики, отзывы

3. **Корзина:**
   - Добавьте товары в корзину
   - Измените количество товаров
   - Перейдите к оформлению заказа

4. **Оформление заказа:**
   - Заполните контактную информацию
   - Выберите способ доставки и оплаты
   - Подтвердите заказ

5. **История заказов:**
   - После оформления заказа вы будете перенаправлены на страницу заказа
   - Для просмотра истории заказов нужно войти в систему
